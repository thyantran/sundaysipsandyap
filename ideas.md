## Idea A — "Editorial Stationery"

**Design Movement:** Contemporary Editorial / Stationery Aesthetic

**Core Principles:**
1. Asymmetric editorial layout — text and image blocks offset from each other, never centered in unison
2. Generous white space as a luxury signal
3. Typographic hierarchy as the primary visual driver
4. Tactile, paper-like feel through subtle texture

**Color Philosophy:**
- Background: near-white sage tint (#f5f7f4) — like unbleached paper
- Primary text: deep forest green (#2d4a28)
- Accent: warm cream (#f0e9d6) for card backgrounds
- CTA: deep forest green button with cream text
- Emotional intent: calm, considered, like a beautiful notebook you don't want to put down

**Layout Paradigm:**
- Full-width hero with product title typeset in oversized serif, left-aligned, with product image floating right
- Features section: alternating text-left / image-right columns
- Marquee ticker of "sunday sips & yap" repeating (mimicking the cover pattern)
- CTA section: full-width sage green band with centered buy button

**Signature Elements:**
1. Repeating diagonal "sunday sips & yap" text marquee (from product cover)
2. Hand-drawn iced coffee cup SVG illustration (from product cover)
3. Serif + italic script font pairing (Playfair Display + Cormorant Italic)

**Interaction Philosophy:**
- Hover on CTA: subtle lift shadow + slight scale
- Scroll-triggered fade-in for each section
- Marquee runs continuously, pauses on hover

**Animation:**
- Entrance: sections fade up (translateY 20px → 0, opacity 0 → 1) on scroll
- Marquee: smooth infinite horizontal scroll
- CTA button: scale(1.03) on hover with 200ms ease

**Typography System:**
- Display: Playfair Display Bold — for "SUNDAY SIPS" hero title
- Script accent: Cormorant Garamond Italic — for "& yap" and subheadings
- Body: DM Serif Text Regular — for descriptions
- Label: DM Sans — for small caps labels and nav

## Idea B — "Zine Collage"

**Design Movement:** DIY Zine / Scrapbook Aesthetic

**Core Principles:**
1. Deliberate imperfection — slightly rotated elements, torn-edge dividers
2. Layered composition — elements overlap each other like cut-outs
3. Handmade warmth through texture and organic shapes
4. Playful but not childish

**Color Philosophy:**
- Background: warm off-white (#faf8f4)
- Sage green (#4a6741) as the dominant accent
- Dusty rose (#d4a5a0) as a secondary warm accent
- Emotional intent: nostalgic, personal, like a scrapbook your best friend made you

**Layout Paradigm:**
- Hero: large product image rotated 2° with title overlapping it
- Features: scattered card layout, each card slightly rotated differently
- Journal prompts section: sticky-note style cards
- CTA: big stamp-style button

**Signature Elements:**
1. Torn paper edge SVG dividers between sections
2. Slightly rotated cards and images
3. Handwritten-style accent font

**Interaction Philosophy:**
- Cards straighten on hover (rotate back to 0°)
- Stamp CTA has ink-press animation on click

**Animation:**
- Cards: rotate to 0 on hover, 300ms ease
- Hero image: gentle float animation (translateY ±6px, 4s infinite)

**Typography System:**
- Display: Abril Fatface — bold, editorial
- Script: Caveat — handwritten feel
- Body: Lato Regular

## Idea C — "Soft Botanical Editorial" ← SELECTED

**Design Movement:** Soft Botanical / Quiet Luxury Editorial

**Core Principles:**
1. Restraint and intentionality — every element earns its place
2. Vertical rhythm through generous line-height and section spacing
3. The product is the hero — everything else serves to frame it
4. Feminine without being fussy — elegant, not cutesy

**Color Philosophy:**
- Background: pale sage (#eef1ec) — the exact tint from the product cover
- Deep forest green (#3a5c35) for all primary text and UI elements
- White (#ffffff) for card surfaces
- CTA: forest green (#3a5c35) with white text — confident, clean
- Emotional intent: Sunday morning calm, the feeling of a beautiful slow day

**Layout Paradigm:**
- Asymmetric hero: large left-side typographic block + right-side product mockup image
- Scrolling marquee ticker below hero (product name repeating)
- Three-column feature cards with icons
- Full-bleed sage green CTA band at bottom
- Sticky minimal nav at top

**Signature Elements:**
1. Continuous "sunday sips & yap" marquee strip (direct reference to product cover)
2. Iced coffee cup line illustration as a decorative motif
3. Mix of uppercase serif for titles and italic script for "&yap" accents

**Interaction Philosophy:**
- Everything feels gentle — no jarring transitions
- CTA button has a warm hover state (slightly lighter green)
- Scroll animations are subtle fade+rise, never dramatic

**Animation:**
- Hero text: staggered word-by-word fade-in on load
- Marquee: continuous smooth scroll, 30s loop
- Feature cards: fade up on scroll intersection
- CTA button: background lightens + subtle shadow on hover

**Typography System:**
- Display: Playfair Display Bold — "SUNDAY SIPS" hero
- Script italic: Cormorant Garamond Italic — "& yap" and pull quotes
- Body: Lora Regular — warm, readable serif for descriptions
- UI labels: DM Sans — clean, modern contrast

## Selected Design: Idea C — "Soft Botanical Editorial"

Rationale: Most faithful to the product's existing visual identity (sage green, serif+script typography, the iced coffee illustration). Feels premium and intentional without being over-designed. The asymmetric layout avoids generic centered AI-slop while keeping the one-page structure simple and scannable.
